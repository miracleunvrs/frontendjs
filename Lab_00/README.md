# Lab 0 — Введение в React + TypeScript

</div>

## Описание

Вводная лабораторная работа для знакомства с основами React и TypeScript. В этой лабораторной работе вы создадите базовые функциональные компоненты, научитесь работать с props, типизацией и inline-стилями.

## Цели обучения

- Создание функциональных компонентов в React
- Работа с TypeScript и типизацией props
- Применение условной логики в компонентах
- Стилизация компонентов с помощью inline-стилей
- Понимание структуры React-проекта с Vite

## Задания

### 1. Компонент Greeting

**Файл:** `src/components/Greeting.tsx`

Создайте компонент приветствия, который:
- Определяет текущее время суток
- Отображает соответствующее приветствие (Good Morning, Good Afternoon, Good Evening)
- Меняет цветовую схему в зависимости от времени суток
- Использует inline-стили для оформления

**Ключевые концепции:**
- Работа с объектом `Date`
- Условная логика с `switch`
- Inline-стили в React
- TypeScript типизация

### 2. Компонент ProfileCard

**Файл:** `src/components/ProfileCard.tsx`

Создайте карточку профиля пользователя, которая:
- Отображает аватар, имя и биографию
- Имеет кнопку действия (Follow)
- Использует типизированный объект данных
- Применяет стилизацию для создания карточного макета

**Ключевые концепции:**
- Работа с объектами данных
- Импорт и использование изображений
- Типизация данных в TypeScript
- Inline-стили для layout

## Структура проекта

```
lab-01/
├── src/
│   ├── components/
│   │   ├── Greeting.tsx      # Компонент приветствия
│   │   └── ProfileCard.tsx   # Компонент карточки профиля
│   ├── assets/               # Статические файлы (изображения)
│   ├── App.tsx              # Главный компонент
│   ├── App.css              # Стили приложения
│   └── main.tsx             # Точка входа
├── public/                  # Публичные файлы
├── index.html
├── package.json
├── tsconfig.json            # Конфигурация TypeScript
└── vite.config.ts           # Конфигурация Vite
```

## Запуск проекта

```bash
# Переход в директорию проекта
cd Lab_0/lab-01

# Установка зависимостей
pnpm install

# Запуск dev-сервера
pnpm dev
```

Приложение будет доступно по адресу `http://localhost:5173`
