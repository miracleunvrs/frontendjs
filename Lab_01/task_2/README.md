## Описание

Практическое задание по работе с состоянием (state) в React с использованием хука `useState`. В этом задании создается простой интерактивный счетчик, который демонстрирует основы управления состоянием компонента.

## Цели обучения

- Понимание концепции состояния (state) в React
- Работа с хуком `useState`
- Обработка событий (onClick)
- Обновление состояния компонента
- Создание интерактивных компонентов

## Задание

### Компонент Counter

**Файл:** `src/components/Counter.jsx`

Создайте компонент счетчика, который:
- Использует хук `useState` для управления состоянием
- Отображает текущее значение счетчика
- Имеет кнопки для увеличения и уменьшения значения
- Обновляет UI при изменении состояния

#### Ключевые концепции:

**1. Хук useState:**
```jsx
const [count, setCount] = useState(0);
```
- `count` — текущее значение состояния
- `setCount` — функция для обновления состояния
- `0` — начальное значение

**2. Обработчики событий:**
```jsx
const handleIncrement = () => {
    setCount(count + 1);
};
```

**3. Привязка обработчиков:**
```jsx
<button onClick={handleIncrement}>+</button>
```

## Как это работает

1. **Инициализация состояния:**
   - При первом рендере компонента `useState(0)` создает переменную состояния `count` со значением 0

2. **Отображение значения:**
   - Текущее значение `count` отображается в элементе `<h2>`

3. **Обработка кликов:**
   - При клике на кнопку "+" вызывается `handleIncrement`
   - При клике на кнопку "-" вызывается `handleDecrement`

4. **Обновление состояния:**
   - Функция `setCount` обновляет значение `count`
   - React автоматически перерисовывает компонент с новым значением
