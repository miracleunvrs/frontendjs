## Ключевые концепции

### Императивный vs Декларативный

**Императивный подход:**
```javascript
// Описываем КАК изменить UI
$('#button').click(function() {
  $('#text').toggleClass('highlight');
});
```

**Декларативный подход:**
```jsx
// Описываем ЧТО должно быть отображено
const [isHighlighted, setIsHighlighted] = useState(false);

return (
  <p className={isHighlighted ? 'highlight' : ''}>
    Текст
  </p>
);
```

### Хук useState

```jsx
const [count, setCount] = useState(0);
```

- `count` — текущее значение состояния
- `setCount` — функция для обновления состояния  
- `0` — начальное значение

### Обработка событий

```jsx
const handleClick = () => {
  setCount(count + 1);
};

<button onClick={handleClick}>Click</button>
```

### Условный рендеринг

```jsx
<p className={condition ? 'classA' : 'classB'}>
  Текст
</p>
```
